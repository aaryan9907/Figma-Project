# Programming Pathshala Landing Page 🚀

A fully responsive landing page inspired by Programming Pathshala's
website design.

This project was built using pure HTML and CSS with a strong focus on: -
Clean layout structure - Pixel-perfect UI - Responsive design - Custom
components - Modern shadows and depth effects

------------------------------------------------------------------------

## 📌 Features

### ✅ Hero Section

-   Custom background positioning
-   Soft shadow CTA buttons
-   Fully centered layout
-   Responsive design

### ✅ Navbar

-   Structured navigation links
-   Styled CTA buttons
-   Mobile-ready layout
-   Custom arrow indicators

### ✅ Course Curriculum Section

-   Responsive grid layout
-   5 visible cards on mobile
-   4 cards per column on desktop
-   "Show More" button with custom double-arrow

### ✅ Build Section

-   Gradient background
-   Interactive card design
-   Custom toggle UI
-   AI Agent styled header
-   Code preview panel

### ✅ Footer Section

-   4-column layout (desktop)
-   Mobile stacked layout
-   Custom social media icons
-   Contact info with image icons
-   Styled spacing and alignment

------------------------------------------------------------------------

## 🛠 Tech Stack

-   HTML5
-   CSS3
-   Flexbox
-   CSS Grid
-   Responsive Media Queries

No JavaScript used.
